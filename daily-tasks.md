## Python лабораторные работы

### Лаба 1

#### Задание 1
- [ ] Создать базовый класс `Node`, в нем должны быть определены:
  - поле с данными `data`
  - поле со списком ссылок на объекты типа `Node` (`link_list`)
- [ ] Написать метод `__init__`:
  - работающий по умолчанию без параметров (присваивает `0` и пустой список полям `data` и `link_list`)
  - с параметрами: число и список ссылок на объекты типа `Node` или число и одна ссылка на объект типа `Node`
- [ ] Написать функцию `clear()`, очищающую значение `data` и присваивающую `link_list` ссылку на пустой список.

#### Задание 2
1. **На базе Node сформировать класс для реализации односвязных списков** (наследование не всегда целесообразно) и реализовать функции:
   - [ ] Добавление элемента в начало списка
   - [ ] Добавление элемента в конец списка
   - [ ] Добавление элемента после i-го элемента
   - [ ] Добавление элемента перед i-м элементом

2. **На базе Node сформировать класс для реализации двусвязных списков**:
   - [ ] Добавление элемента в начало списка
   - [ ] Добавление элемента в конец списка
   - [ ] Добавление элемента после i-го элемента
   - [ ] Добавление элемента перед i-м элементом

3. **На базе Node сформировать класс для реализации стека**:
   - [ ] Реализовать функции добавления и удаления элемента
   - [ ] Реализовать функцию, формирующую новую очередь из списка очередей

4. **На базе Node сформировать класс для реализации очереди**:
   - [ ] Реализовать функции добавления и удаления элемента
   - [ ] Реализовать функцию, формирующую новый стек из стеков

#### Задание 3
- [ ] Написать свой собственный класс обработки ошибок, унаследованный от общего класса ошибок, с переписанными необходимыми функциями.

**Примечания:**
- Классы вынести в отдельную библиотеку (каждый класс по возможности внутри отдельного файла внутри папки, создать файл `__init__.py`).
- В основной программе привести пример с демонстрацией всех функций и пример обработки ошибки.

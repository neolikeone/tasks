Задание 1.
    • Создать базовый класс Node, в нем должны быть определены поля с данными (поле data)  и поле со списком ссылок объектов типа Node (поле link_list).
    • Для этого класса написать метод __init__ (работающий как по умолчанию без параметров и присваивающий 0 и пустой список полям data и link_list , так и с параметрами,  число и список ссылок на объекты типа Node или число и одна ссылка на объект типа Node)
    • Написать ф-цию clear(), очищающий значение data и присваивающий link_list ссылку на пустой список.
Задание 2
    1. На базе Node сформировать класс для реализации односвязных списков (наследование не всегда целесообразно) и реализовать ф-ции добавления и удаления эл-та.

    1. Добавление эл-та в начало списка
    2. Добавление эл-та в конец списка
    3. Добавление эл-та после i-го элемента 
    4. Добавление эл-та перед i-го элементом
------------------
    2. На базе Node сформировать класс для реализации двусвязных списков.

    1. Добавление эл-та в начало списка
    2. Добавление эл-та в конец списка
    3. Добавление эл-та после i-го элемента 
    4. Добавление эл-та перед i-го элементом
-----------------
    3. На базе Node сформировать класс для реализации стека 
        1. Реализовать ф-ции добавления и удаления эл-та
        2. Реализовать ф-цию, формирующую новую очередь из списка очередей

    4. на базе Node сформировать класс для реализации очереди 
        1. реализовать ф-ции добавления и удаления эл-та
        2. реализовать ф-цию, формирующую новый стек из стеков
